Les deux axes de la nomenclature

Le budget est classe selon deux axes : la destination (« a quoi sert la depense ? ») et la nature (« quel type de depense ? »).

DESTINATION A quoi sert la depense ?

  • MIN Ministere
    17 ministeres — code numerique (1-70)
  • MSN Mission
    ~46 missions — code 2 lettres (AA-ZF)
    • PGM Programme
      ~181 programmes — code 3 chiffres (101-878)
      • ACT Action
        ~697 actions — code PGM-NN
        • SSA Sous-action
          ~289 sous-actions — code PGM-NN-NN

NATURE Quel type de depense ?

  • TTR Titre
    7 titres — code numerique (1-7)
    • CAT Categorie
      19 categories — code numerique (10-73)
Le croisement des deux axes donne le detail complet : par ex. les depenses de personnel (nature) du programme Justice judiciaire (destination).

Le role pivot du Programme

Le Programme fait le pont entre Mission et Ministere, et se decline en Actions / Sous-actions.

Exemple : suivre un credit

Credits de personnel diplomatique du ministere des Affaires etrangeres :

BG Budget general
→
MIN 1 Affaires etrangeres
→
MSN AA Action exterieure
→
PGM 105 Action de la France
→
ACT 01 Coordination diplomatique
→
SSA 01 Etat-major : cabinets

Croise avec TTR 1 Depenses de personnel → CAT 10 Remunerations, en distinguant AE (engagements) et CP (paiements).

Points cles

  • 1. Les codes sont contextuels par annee : le programme 104 en 2019 ≠ le meme en 2024.
  • 2. Le Programme (PGM) est le noeud central : il relie mission, ministere, actions et sous-actions.
  • 3. Mission et Ministere ne sont pas directement lies : c'est le programme qui fait le pont.
  • 4. Les Titres/Categories (nature) sont des invariants stables, contrairement aux nomenclatures de destination.
  • 5. AE vs CP : AE = promesses, CP = paiements. Un investissement pluriannuel a des AE concentrees et des CP etalees.
